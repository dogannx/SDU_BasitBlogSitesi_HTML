Biyografi

Merhaba, ben Doğan Coşman. Bilgisayar Mühendisliği 3. sınıf öğrencisiyim. Yazılım geliştirme alanına özellikle web teknolojileri üzerine ilgi duyuyorum.

Şu ana kadar hem kişisel projelerimde hem de ekip çalışmaları içinde deneyimler kazandım. Özellikle Go (Golang), JavaScript ve modern web frameworkleri ile çalışıyorum. Backend geliştirme konusunda Go, Bun, Fiber, PostgreSQL gibi teknolojilerle projeler geliştiriyorum. Frontend tarafında ise Vue.js ve TypeScript ile çalışmayı seviyorum.

Yeteneklerim

  • Web Geliştirme: Go, JavaScript, Vue.js, Fiber
  • Veritabanı Yönetimi: PostgreSQL, Gorm
  • Versiyon Kontrol: Git & GitHub
  • Yazılım Mimarisi ve Proje Yönetimi
  • Problem çözme ve algoritma geliştirme

Şu Ana Kadar Yaptıklarım

  • Kişisel Blog Projesi: Go ve PostgreSQL kullanarak backend tarafını geliştirdim.

  • Teknofest Projesi: Eğitim alanında VR laboratuvar, yapay zekâ destekli koçluk ve bulmaca tabanlı öğrenme içeren web platformunun proje kaptanlığını üstlendim.
